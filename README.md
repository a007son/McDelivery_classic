# McDelivery Classic

「讓金閃閃再次偉大！」
經典線上菜單介面又回來了 — 重新整理、全新上架，讓你隨時輕鬆查價不痛苦

## 關於這個專案
本專案整理台灣麥當勞公開的產品及價格資訊並參考舊版歡樂送菜單（早餐時段 + 一般時段），以靜態頁面的形式呈現。

## 功能特色
- 全時段展示菜單：早餐時段及一般時段
- 菜單資訊（品項名稱、價格）皆來自公開資料來源
- 以現代前端技術構建（Vite + React + Tailwind）

## 授權與使用限制
本專案由Gemini 3 PRO建構，採 **自訂授權條款（禁止改作／禁止再製／禁止商用）**。
使用本站資料僅限個人用途，禁止複製、修改或以任何形式再散布。

## 注意事項
- 本專案非官方麥當勞網站，亦非麥當勞授權發布之內容。
- 圖片與價格由公開管道蒐集，因時間、活動或地區變動，標示之售價或品項可能與實際情況有所不同。
- 若官方公開完整菜單或恢復服務，本站將停止更新。
